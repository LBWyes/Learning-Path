# 第二次实验文档
- 使用`Link()`无参构造，会生成一个头节点
- 使用`Link(T d)`进行带数据的有参构造，会生成一个头节点和紧跟在头节点之后的数据节点
- `~Link()`析构函数
- `add(T d)`方法在链表尾部直接添加数据d
- `add(T d,int n)`方法在位置`n`处添加数据d
- `del()`方法删除末尾的一个节点
- `del(int n)`方法删除位置`n`处的一个节点
- `upadte(T d,int n)`方法更新位置`n`处的数据
- `isempty()`方法判断链表是否为空
- +运算符重载用于将两个链表顺次相连
- 运算符重载用于删除位置`n`处的数据
- [] 运算符重载用于访问位置`n`处数据
